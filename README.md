## 项目介绍
- 项目名称：gin-blog
- 项目简介：
  - 这是一个基于Gin框架的博客项目，用于展示和管理博客文章。
  - 项目使用Gin作为Web框架，提供了用户注册、登录、文章管理等功能。
  - 项目使用了GORM作为ORM框架，用于操作数据库。

## 功能介绍

- logrus日志初始化
- 数据库读写
- 根据ip获取地理位置
